TypeName

ClosingGenericBracketsMustBeSpacedCorrectly

CheckId

SA1015

Category

Spacing Rules

Cause

A closing generic bracket within a C# element is not spaced correctly.

Rule Description

A violation of this rule occurs when the spacing around a closing generic bracket is not correct.

A closing generic bracket should never be preceded by whitespace, unless the bracket is the first character on the line. A closing generic bracket should be followed by an open paranthesis, an end of line or a single whitespace (but not whitespace and an open paranthesis).

How to Fix Violations

To fix a violation of this rule, ensure the whitespace around the closing generic bracket is correct.
